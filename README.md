# SubTrack

> 个人订阅预算追踪小程序 - 清晰管理订阅支出，智能控制预算

## 📖 项目简介

SubTrack 是一款专注于软件订阅服务管理的个人财务管理类小程序。帮助用户追踪和管理各类软件订阅，清晰了解每月/每年的订阅支出，有效设定和控制预算。

### 核心价值

- **清晰透明**：直观展示订阅支出全貌
- **预算控制**：帮助用户设定和监控预算
- **智能提醒**：续费日期提前通知，避免意外扣费
- **省心省钱**：减少订阅管理的认知负担

## 🎯 核心功能

### 订阅管理
- ✅ 添加/编辑/删除订阅
- ✅ 多币种支持（CNY/USD/EUR/GBP/JPY/HKD/TWD）
- ✅ 自定义扣款日期规则（月付/季付/年付）
- ✅ 订阅状态管理（有效/已取消）
- ✅ 订阅分类与备注

### 预算管理
- ✅ 月度预算设置
- ✅ 实时预算进度追踪
- ✅ 超支警告提示
- ✅ 多币种汇率转换（用户手动设置）
- ✅ 未来费用预览

### 数据可视化
- ✅ 费用分布饼图
- ✅ 预算进度条（动态颜色标识）
- ⏳ 月度支出趋势图
- ⏳ 分类统计图表

### 续费提醒
- ✅ 即将续费列表展示
- ⏳ 小程序内提醒卡片
- ⏳ 订阅消息推送（需用户授权）

### 数据管理
- ⏳ JSON 数据导出
- ⏳ JSON 数据导入
- ⏳ 数据备份与迁移

## 🛠 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | weapp-vite | 基于 Vite 构建原生小程序开发工具 |
| UI组件库 | 自定义组件 | 微信小程序原生组件库 |
| 数据存储 | 微信小程序本地存储 | 数据完全存储在本地，无需云端服务 |
| 图表库 | ECharts for WeChat | 微信小程序图表组件 |

## 🚀 快速开始

### 环境要求

- Node.js >= 16
- npm >= 7
- 微信开发者工具

### 安装依赖

```bash
cd wechat
npm install
```

### 开发模式

```bash
npm run dev
```

### 编译构建

```bash
npm run build
```

### 使用微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `wechat/miniprogram` 目录
3. 开启"不校验合法域名"（本地开发）
