# 开发笔记

本目录用于记录开发过程中的技术难点、错误和解决方案。

## 目录结构

```
devnotes/
├── README.md          # 本文件
├── pitfalls/          # 踩坑记录
├── snippets/          # 常用代码片段
└── references/        # 参考资料
```

## 文档命名规范

```
YYYYMMDD_简短描述.md

例如：
20260319_weapp_vite_build_error.md
20260320_uniapp_page_lifecycle.md
```
