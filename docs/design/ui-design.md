## 3. 核心组件

### 3.1 预算进度组件 (BudgetProgress)

**Props:**
- `spent`: 已支出金额
- `budget`: 预算金额
- `currency`: 货币符号

**状态颜色:**
- 使用率 < 80%: 绿色
- 80% ≤ 使用率 ≤ 100%: 黄色
- 使用率 > 100%: 红色

### 3.2 订阅卡片组件 (SubscriptionCard)

**Props:**
- `icon`: 图标
- `name`: 订阅名称
- `amount`: 金额
- `currency`: 货币
- `cycle`: 计费周期
- `nextBillingDate`: 下次扣款日期

### 3.3 金额输入组件 (AmountInput)

**Props:**
- `value`: 金额
- `currency`: 货币代码
- `onChange`: 值变更回调

## 4. 交互规范

### 4.1 手势操作

- **下拉刷新**：列表页刷新数据
- **左滑删除**：订阅卡片删除订阅
- **长按**：订阅卡片快捷操作菜单

### 4.2 反馈提示

- **保存成功**：Toast 提示
- **删除确认**：Modal 弹窗
- **网络错误**：Toast 提示 + 重试按钮
- **加载中**：Loading 动画

### 4.3 空状态

**无订阅时：**
```
      📭
  还没有订阅记录
  点击下方按钮添加

   [+ 添加订阅]
```

**无即将续费：**
```
      ✅
未来7天没有续费计划
```
