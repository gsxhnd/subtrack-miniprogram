# SubTrack 产品概览

## 项目定位

SubTrack 是一个面向微信小程序的订阅管理工具，核心目标是帮助用户看清订阅支出、控制预算、减少无效续费。

**一句话定位：** 个人订阅管理 + 预算追踪助手。

## 目标用户

- 有多个数字服务订阅（5+）的用户
- 希望控制每月固定支出的用户
- 经常忘记续费日期或忘记取消订阅的用户
- 需要统一管理多币种订阅的用户

## 核心价值

- **支出透明**：统一展示月度/年度订阅总支出
- **预算可控**：提供预算设置、进度可视化与超支预警
- **管理省心**：集中管理订阅状态与续费节奏
- **数据可迁移**：支持从本地使用逐步过渡到云端同步

## 版本现状

- **已完成（v1.x）**：本地存储、订阅 CRUD、预算管理、基础统计
- **进行中（v2.0）**：CloudBase + MySQL 云端化与多设备同步
- **后续规划（v2.1+）**：高级统计、提醒增强、家庭共享等
