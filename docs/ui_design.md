# SubTrack 界面设计文档

## 1. 页面导航结构

### 1.1 整体架构

```
┌─────────────────────────────────────────┐
│              TabBar 导航                │
├─────────┬─────────┬─────────┬──────────┤
│  首页   │ 订阅列表 │  统计   │   设置   │
└────┬────┴────┬────┴────┬────┴────┬─────┘
     │         │          │         │
     │         │          │         └── 设置页
     │         │          │
     │         │          └── 统计页
     │         │              ├── 月度趋势图
     │         │              └── 分类统计
     │         │
     │         └── 订阅列表页
     │              ├── 添加订阅按钮
     │              └── 订阅卡片列表
     │                   └── 点击进入详情
     │                        └── 编辑/删除
     │
     └── 首页仪表盘
          ├── 预算进度卡片
          ├── 费用分布饼图
          └── 即将续费列表
```

### 1.2 页面列表

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | /pages/index/index | 仪表盘，展示概览信息 |
| 订阅列表 | /pages/subscription/index | 所有订阅列表 |
| 订阅详情 | /pages/subscription-detail/index | 单个订阅详情 |
| 订阅编辑 | /pages/subscription-edit/index | 添加/编辑订阅 |
| 统计 | /pages/statistics/index | 数据统计分析 |
| 设置 | /pages/settings/index | 用户设置 |
| 汇率设置 | /pages/exchange-rate/index | 手动设置汇率 |
| 导出数据 | /pages/export/index | 导出数据为 JSON |
| 导入数据 | /pages/import/index | 从 JSON 导入数据 |

**注意：** 以上路由为微信小程序页面路径，位于 `wechat/src/pages/` 目录下。

```

├────────────────────────────────────┤

│  └──────────────────────────────┘  │
│                                    │
│  ┌──────────────────────────────┐  │
│  │    费用分布                   │  │
│  │         ┌─────┐              │  │
│  │       ╱   饼图   ╲            │  │
│  │      │           │           │  │
│  │       ╲         ╱            │  │
│  │         └─────┘              │  │
│  │  ChatGPT 35%  Notion 25%...  │  │
│  └──────────────────────────────┘  │

│  │ 🤖 ChatGPT Plus   ¥120  3天后│  │
│  ├──────────────────────────────┤  │
│  │ 📝 Notion         ¥80   5天后│  │
│  ├──────────────────────────────┤  │
│  │ 🎵 网易云音乐     ¥15   7天后│  │
│  └──────────────────────────────┘  │
│                                    │
└────────────────────────────────────┘
```

### 2.2 组件说明

| 区域 | 组件 | 功能 |
|------|------|------|
| 顶部 | 预算进度卡片 | 展示本月支出/预算，进度条可视化 |
| 中部 | 费用分布饼图 | 展示各订阅费用占比 |
| 底部 | 即将续费列表 | 未来7天内需扣费的订阅 |

### 2.3 交互说明

- 点击预算进度卡片 → 跳转到设置页（预算设置）
- 点击饼图扇区 → 高亮显示该订阅详情
- 点击即将续费项 → 跳转到订阅详情页

**筛选选项：**
- 全部

- 已取消订阅
- 按分类筛选

**排序选项：**
- 按名称（A-Z）
- 按金额（高到低/低到高）
- 按下次扣款日期（近到远）

## 4. 订阅编辑页设计

### 4.1 页面原型

```
┌────────────────────────────────────┐
│  ← 添加订阅                   [保存]│
├────────────────────────────────────┤
│                                    │
│  订阅名称 *                        │
│  ┌──────────────────────────────┐  │
│  │ 请输入订阅名称                │  │
│  └──────────────────────────────┘  │
│                                    │
│  金额与货币 *                      │
│  ┌──────────────────────────────┐  │
│  │ ¥ 0.00          [CNY ▼]      │  │
│  └──────────────────────────────┘  │
│                                    │
│  计费周期 *                        │
│  ○ 月付  ○ 季付  ○ 年付           │
│                                    │
│  扣款日期 *                        │
│  ┌──────────────────────────────┐  │
│  │ 每月  [15] 号扣款             │  │
│  └──────────────────────────────┘  │
│  或（年付时）                      │
│  ┌──────────────────────────────┐  │
│  │ 每年 [3] 月 [15] 日扣款       │  │
│  └──────────────────────────────┘  │
│                                    │
│  开始日期                          │
│  ┌──────────────────────────────┐  │
│  │ 2024-03-01            [选择] │  │
│  └──────────────────────────────┘  │
│                                    │
│  分类                              │
│  ┌──────────────────────────────┐  │
│  │ 请选择分类            [选择] │  │
│  └──────────────────────────────┘  │
│                                    │
│  备注                              │
│  ┌──────────────────────────────┐  │
│  │ 可选，如用途、账户信息        │  │
│  └──────────────────────────────┘  │
│                                    │
│            [删除订阅]              │
│                                    │
└────────────────────────────────────┘
```

### 4.2 表单字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 订阅名称 | 文本输入 | 是 | 如：ChatGPT Plus |
| 金额 | 数字输入 | 是 | 订阅费用 |
| 货币 | 下拉选择 | 是 | CNY/USD/EUR等 |
| 计费周期 | 单选 | 是 | 月付/季付/年付 |
| 扣款日期 | 复合选择 | 是 | 根据周期动态展示 |
| 开始日期 | 日期选择 | 否 | 订阅开始时间 |
| 分类 | 下拉选择 | 否 | AI工具/效率工具等 |
| 备注 | 文本域 | 否 | 备注信息 |

### 4.3 扣款日期选择逻辑

**月付模式：**
```
每月 [1-31] 号扣款
```

**季付模式：**
```
每季度 [1-31] 号扣款
（从开始日期算起，每3个月扣款一次）
```

**年付模式：**
```
每年 [1-12] 月 [1-31] 日扣款
```

## 10. 通用组件设计

### 10.1 预算进度组件 (BudgetProgress)

```
┌────────────────────────────────────┐
│  本月支出                          │
│  ¥ 128.00 / ¥ 200.00              │
│  ████████░░░░░░░░ 64%             │
└────────────────────────────────────┘
```

**Props:**
- `spent`: 已支出金额
- `budget`: 预算金额
- `currency`: 货币符号

**状态颜色:**
- 使用率 < 80%: 绿色
- 80% ≤ 使用率 ≤ 100%: 黄色
- 使用率 > 100%: 红色

### 10.2 订阅卡片组件 (SubscriptionCard)

```
┌────────────────────────────────────┐
│ [图标]  订阅名称                   │
│         ¥金额/周期  ·  状态信息    │
└────────────────────────────────────┘
```

**Props:**
- `icon`: 图标URL
- `name`: 订阅名称
- `amount`: 金额
- `currency`: 货币
- `cycle`: 计费周期
- `status`: 状态
- `nextBillingDate`: 下次扣款日期
- `cancelledAt`: 取消日期

### 10.3 金额输入组件 (AmountInput)

```
┌────────────────────────────────────┐
│ ¥ [金额输入框]      [货币选择 ▼]   │
└────────────────────────────────────┘
```

**Props:**
- `value`: 金额
- `currency`: 货币代码
- `onChange`: 值变更回调

### 10.4 扣款日期选择组件 (BillingDatePicker)

```
月付模式:
┌────────────────────────────────────┐
│ 每月 [ 15 ] 号扣款                 │
└────────────────────────────────────┘

年付模式:
┌────────────────────────────────────┐
│ 每年 [ 3 ] 月 [ 15 ] 日扣款        │
└────────────────────────────────────┘
```

**Props:**
- `cycle`: 计费周期
- `day`: 扣款日
- `month`: 扣款月份（年付时）
- `onChange`: 值变更回调
