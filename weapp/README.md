# weapp-vite-template

`weapp-vite` 模板

## 使用方式

### 开发

- `npm dev`

- `npm dev --open` 可以打包并直接启动微信开发者工具

### 构建

`npm build`

### 打开微信开发者工具

`npm open`

### 生成组件/页面

`npm g path/to/your/component`
